# `@tubstrr/prettier-config`

> My personal [Prettier](https://prettier.io) config.

## Usage

**Install**:

```bash
$ yarn add --dev @tubstrr/prettier-config
```

**Edit `package.json`**:

```jsonc
{
  // ...
  "prettier": "@tubstrr/prettier-config"
}
```
